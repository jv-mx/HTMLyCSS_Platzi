Hermano adyacente
(Adjacent sibling)

Este combinador aplica la regla de estilo al elemento que se encuentre mas cercano a su referente.
En el siguiente ejemplo a todas las etiquetas p cercanas a un h2.

h2 + p {
color: red;
}

Hijo Directo
(Child)

Este combinador aplica la regla de estilo a los elementos que sean hijos directos del objeto indicado.
Se expresa de la siguiente manera:

div > p {
color: red;
}
